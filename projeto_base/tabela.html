Regra CSS Descrição
.setor Responsavel por ocupar 100 % da pagina
.container Define uma delimitação de largura de 1000 pixels e centraliza o conteúdo da pagina. Usando para manter o estilo do site, independetemente do tamanho do monitor.
.coluna mostra como bloco e define o comportamento para se ajustar usando flutuação a esquerda, fazendo com que as colunas se ajustem uma ao lado da outra
.l-25 Define a largura para 25% do tamanho delimitado pelo container. Utilizado juntamente com a classe .coluna
.l-33 Define a largura para 33% do tamanho delimitado pelo container. Utilizado juntamente com a classe .coluna
.l-50 Define a largura para 50% do tamanho delimitado pelo container. Utilizado juntamente com a classe .coluna
.l-100 Define a largura para 100% do tamanho delimitado pelo container. Utilizado juntamente com a classe .coluna
.a-100px Define a altura para 100 pixels
