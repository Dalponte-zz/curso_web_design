Fundamentos de HTML 5

EU acho que o HTML é um mal necessário. Até porque, o XTML que seria a alternativa, é um inferno.

A classe .section define a estilização para espaçamento interno e tamanho de fonte.
A classe .bg-branco vai definir qual é a cor de fundo da secção

O motivo é uma questão de produtividade Definir regras nas classes, de forma que possam ser utilizadas novamente ao longo do código.

Conteúdos da prova

  • Saber o que é:
    • Tag
    • Atributo da tag
    • Conteúdo da tag
    • Regra CSS
    • Propriedade CSS
    • Valor da propriedade
    • Declaração (abertura) da tag
    • Finalização da tag
  • Saber Como faz:
    • Fundo com cores
    • Fundo com imagens
    • Mostrar como bloco (display:block)
    • alinhar bloco a direita (display, float)
    • Tamanho (width, height, padding, margin)
    • Unidades de medida para tamanho (px ou %)
    • Espaçamento interno (padding)
    • Espaçamento externo (margin)
    • Tamanho de fonte
    • Largura de fonte
    • Tipo de fonte
    • Alinhamento de fonte
  • Saber o que faz como
    • cabeçalho (head)
    • Corpo (body)
    • headings (h1, h2, h3 ...)
    • paragrafos (p)
    • estilo (b, i, u)
    • Divisor(div)
    • listas (ul e ol com li)
    • Tabelas (table com tr, th e td)
    • Expansão de colunas (atrib. colspan)
    • Expansão de linhas (atrib. rowspan)
